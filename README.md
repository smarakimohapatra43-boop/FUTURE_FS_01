# 🌐 Personal Portfolio Website

A responsive and modern full stack portfolio website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

This website showcases my:
- Skills
- Projects
- Web development journey
- Contact information

It also includes a working contact form with email notifications and MongoDB database integration.
